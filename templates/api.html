API Documentation

Integrate our powerful media downloading capabilities into your own applications.

Video Information

GET /api/info?url={URL}

Returns title, thumbnail, and duration for the media.

YouTube Audio

GET /api/youtube/audio?url={URL}

Returns the audio file directly as a download.

YouTube Video

GET /api/youtube/video?url={URL}

Returns the video file directly as a download.

TikTok Download

GET /api/tiktok/download?url={URL}

Returns the TikTok video without watermark.

Usage Notes

All endpoints return files directly with the appropriate MIME type. Ensure URLs are URL-encoded.
